PETSHOP - 2019

Para la ejecución del software es necesario correr el archivo "Main.py", el cual es la clase principal y conecta todas las demas clases del sistema.
SI SE LLEGA A EJECUTAR OTRA DE LAS CLASES Y NO EL MAIN, EL PROGRAMA FALLARÁ.

--------------------------------------------------------------------------------------------------------------------------------------------------------

Si se desea probar sin necesidad de hacer ningun tipo de registro de productos o algo parecido, se deben agregar los respectivos datos ficticios y se poblará el sistema de datos básicos para comprobar todas las funcionalidades.

--------------------------------------------------------------------------------------------------------------------------------------------------------

A la hora de seleccionar opciones si se le pide ingresar números por favor ingrese estos y no letras, ya que si se ingresa algun string el programa fallará.
